HOW

  1. .scripts/build-publish-backfill.ps1 — PowerShell script that
     parses each .md file (UTF-8), extracts SUBJECT + OPENING + body,
     slugifies, and emits a single SQL transaction with dollar-quoted
     bodies ($body_md$...$body_md$) — handles em-dashes, apostrophes,
     and Markdown without escaping headaches.
  2. .scripts/publish-backfill.sql — the emitted SQL (21 KB, 5
     INSERTs in one transaction). Kept in repo as the audit receipt.
  3. Executed via Supabase MCP execute_sql against project
     orangebox (sthziuzmreqnrxmllodj). All 5 rows committed.
