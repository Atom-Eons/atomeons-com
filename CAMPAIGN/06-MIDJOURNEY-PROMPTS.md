## Workflow

1. Open https://midjourney.com (operator's account, already granted)
2. Paste prompt into "What will you imagine?" search bar
3. Wait for 4-image grid
4. Select best variant, click U1/U2/U3/U4 to upscale
5. Right-click upscaled image → Save As → name it `MJ-N-purpose.png`
6. Save to `C:/AtomEons/.claude/worktrees/bold-leakey-4470e8/CAMPAIGN/assets/`
7. Use in FB/IG/X post via paste
