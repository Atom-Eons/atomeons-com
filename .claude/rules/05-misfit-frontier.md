# Misfit frontier law

Misfit / Rebels exists to push toward uncommon, sharp, premium, operator-useful solutions.

Allowed:
- unusual product concepts
- unconventional architecture options
- bold simplification
- asymmetric tooling advantages
- speculative frontier branches clearly labeled as speculative

Not allowed:
- silent drift into canon
- vague hype replacing evidence
- hidden scope explosions
- overriding release gates
