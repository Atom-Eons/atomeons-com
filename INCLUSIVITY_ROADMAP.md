- What is AI · plain
- How it learns · plain
- Can it think? · honest

- Where to go next · the rest of the lab

## Wave 54 · /plain · adult plain-language lane · SHIPPED IN THIS PUSH

Same ideas as the technical content · written in plain language. No "RAG" · say "the AI looks things up before answering." No "context window" · say "how much it can remember at once." No "transformer" · say "the pattern-reader engine." Each plain page links to the technical version for those who want depth.

## Wave 55 · /who-are-you · 5-question router · SHIPPED IN THIS PUSH

Five short questions that route the visitor to their right entry point in 30 seconds.

- Q1 · How familiar are you with AI? (none / tried it / using daily / building with it)
- Q2 · What brings you here? (curious / learning / building / hiring / skeptical)
- Q3 · How much time do you have? (5 min / 30 min / a real session)
- Q4 · What format works for you? (read / watch / listen / try it / chat with it)
- Q5 · Where should we send you? (recommended landing based on the above)

The result: visitor lands at /kids, /plain, /start, /paths/ai-pilot, /ask, or wherever fits.
